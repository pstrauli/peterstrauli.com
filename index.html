I’m a Frontend Developer, living in Prague, and I’m passionate about web design that is both beautiful and inclusive.

Frontend Development

I’m constantly honing my skill set and staying current with the latest developments in web technologies such as HTML, CSS , JavaScript and React, and understanding to use them best for Search Engine Optimisation and web accessibility. For me, function and beauty should go hand-in-hand. I always consider the User Experience and how to make your site a pleasure to use.

Accessibility and Inclusivity

I approach web design with every user in mind. Considering a mix of abilities, I’m constantly researching the latest best practices in web accessibility, and building robust and usable sites from the ground up. It makes no sense to exclude any potential user from getting the most out of your website and I find this approach leads to sites that are simpler and easier for everyone to use.

Other Skills
